OMI - Учебный онлайн-магазин

Инновационный проект, разрабатываемый в рамках изучения современных веб-технологий и фреймворка Spring

О проекте

OMI (Online Market Internship) - это полнофункциональный учебный онлайн-магазин, созданный в качестве дипломного проекта для демонстрации навыков разработки современных веб-приложений.

Проект разрабатывается с нуля, начиная от проектирования архитектуры и заканчивая реализацией сложных бизнес-процессов, характерных для электронной коммерции.

Основная цель проекта - создание качественного продукта, который демонстрирует владение современным стеком технологий и лучшими практиками разработки.

Технологический стек

Проект построен на современных и надежных технологиях, обеспечивающих производительность, безопасность и масштабируемость:

Java & Spring

Backend на Spring Boot, Security, Data JPA

PostgreSQL

Надежная реляционная база данных

HTML5 & CSS3

Семантическая верстка и современные стили

JavaScript

Интерактивность и AJAX-запросы

REST API

Архитектура RESTful сервисов

OAuth2

Безопасная аутентификация

Ключевые особенности

  • Полноценная система аутентификации и авторизации
  • Каталог товаров с фильтрацией и поиском
  • Корзина покупок и процесс оформления заказа
  • Личный кабинет пользователя с историей заказов
  • Система скидок и акционных предложений
  • Панель администратора для управления контентом
  • Адаптивный дизайн для всех устройств
  • Оптимизированная производительность

Учебные цели

Этот проект создан в образовательных целях и призван продемонстрировать следующие навыки:

  • Проектирование и разработка полноценного веб-приложения
  • Работа с современными Java-фреймворками (Spring)
  • Создание безопасных и масштабируемых систем
  • Реализация сложных бизнес-процессов
  • Оптимизация производительности приложения
  • Следование принципам чистого кода и лучшим практикам
